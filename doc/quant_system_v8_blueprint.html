♟ MCTS · 蒙特卡洛树搜索详解
动作空间设计
方向: Long / Short / Hold / Close
仓位: {10%, 20%, 30%, 50%} Kelly
每步合法动作 ≤ 16个
奖励函数
R = ŷ · conf - λ·σ - cost
λ=0.3(风险惩罚) · γ=0.95(折现)
cost=spread+commission
UCB1 选择
UCB = Q(s,a) + c·√(ln N/n)
c=1.4(探索系数)
虚损(Virtual Loss)防重复
并行化加速
异步MCTS: 多线程并行rollout · AlphaCast批量推断(batch_size=32) · GPU并行仿真
目标: 1000次仿真 < 200ms · 不阻塞实时数据流 · 仿真结果缓存30s
降级模式
高延迟时自动降为
贪心策略(N=1,M=50)
<50ms 保证响应
# MCTS 主循环（伪代码） def mcts_search(state, alphacast, n_sim=1000, N=10): root = MCTSNode(state) for _ in range(n_sim): node = select(root) # UCB1 selection node = expand(node) # 展开合法动作 value = simulate(node, alphacast, N) # AlphaCast rollout backprop(node, value) # 价值回传 return best_path(root) # 最优路径提取 # simulate: N步AlphaCast快速推断 def simulate(node, model, N): total = 0 for step in range(N): pred = model.fast_infer(node.state) reward = pred.y * pred.conf - λ * pred.sigma total += γ**step * reward return total
